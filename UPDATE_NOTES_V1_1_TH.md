# อัปเดต SJ ทาสีโรยตัว v1.1

รายการที่เพิ่มตามคำขอ:

- เพิ่มระบบปรับแต่งธีม Admin/User ผ่านหน้า `ตั้งค่าเว็บ`
- เพิ่มตั้งค่าโลโก้และ Favicon พร้อมอัปโหลดไฟล์
- เพิ่มระบบเบอร์ติดต่อหลายเบอร์ เพิ่ม/แก้ไข/ลบได้ โดยรองรับใช้งานหลัก 2 เบอร์
- เพิ่มช่องทางติดต่อ Social หลายรายการ
- เพิ่มข้อมูลตัวอย่างสำหรับใบเสนอราคา เช่น ข้อมูลบริษัท เงื่อนไข เลขภาษี ที่อยู่บริษัท
- เปลี่ยนเมนู “กลับหน้าบ้าน” เป็น “ดูหน้าเว็บไซต์”
- เพิ่มเมนู Backup / Export และ Export ข้อมูลเป็น JSON
- เพิ่มรายงานภาพรวมเว็บไซต์ใน Dashboard พร้อมกราฟใบเสนอราคารายเดือน
- เพิ่มไอคอนใน Sidebar Admin
- แก้ CSS ปุ่มให้ตัวหนังสืออยู่กลาง ไม่ชิดด้านบน
- เพิ่มให้ผลงานมีอัลบั้มรูปหลายรูป และผู้ใช้ดูอัลบั้มได้ที่หน้ารายละเอียดผลงาน
- เพิ่มให้บริการสามารถอัปโหลดรูปภาพได้ และแสดงในหน้าบ้าน
- เพิ่มไฟล์ `app/Http/Controllers/Controller.php` เพื่อแก้ Error `Class App\Http\Controllers\Controller not found`

หลังอัปเดตแนะนำรัน:

```bash
composer dump-autoload
php artisan optimize:clear
php artisan storage:link
```

ถ้าใช้ฐานข้อมูลเดิม ไม่จำเป็นต้อง migrate ใหม่ เพราะรอบนี้ใช้ตาราง `site_settings`, `project_images`, `services.image` ที่มีอยู่แล้ว
