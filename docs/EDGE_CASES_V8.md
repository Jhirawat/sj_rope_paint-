# EDGE CASES V8

## Login พนักงาน
- Username ซ้ำไม่ได้
- PIN ต้องเป็นตัวเลข 4-6 หลัก
- ใส่ PIN ผิด 5 ครั้ง ล็อก 15 นาที
- พนักงานลาออก/ระงับ ไม่ควร Login ได้

## เงินเบิก
- เบิก 100 / 200 / 500 / 1000 ได้ตามปกติ
- หากยอดรวมต่อรอบเกิน 1,000 ให้เป็นคำขออนุมัติพิเศษ
- เงินเบิกพิเศษต้องมี Badge และแจ้งเตือน Sidebar
- ไม่ให้ mark paid ถ้ายังไม่ approved

## เข้างาน/แรง
- เข้างานซ้ำวันเดียวกันไม่ได้
- 07:31 เป็นต้นไป = สาย
- ต้องรองรับรูปหาย / ไม่มีรูป ด้วย fallback
- สรุป 0 แรงต้องบันทึกได้
- ขาดงานต้องแยกจากลา

## Dashboard/KPI
- ไม่มีข้อมูลรายเดือนต้องไม่ Error
- Top สาย/Top ขาด/Top เบิก ต้องแสดง empty state ได้
- ข้อมูลพนักงานที่ถูกปิดใช้งานต้องไม่ทำให้หน้า Error

## Theme
- Light/Dark ต้องเปลี่ยนเฉพาะฝั่ง Admin
- Dropdown/Input/Table ใน Dark Mode ต้องอ่านได้

## ป้องกัน Error
- 403 / 404 / 419 / 500 มีหน้า custom
- relation หาย เช่น employee/user/team null ต้องไม่ error
