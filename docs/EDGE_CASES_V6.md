# EDGE CASES V6

## Attendance
1. เข้างานซ้ำในวันเดียวกันต้องไม่ได้
2. ออกงานซ้ำต้องไม่ได้
3. ออกงานโดยไม่มีเข้างานต้องไม่ได้
4. เข้าหลัง 07:30 ต้องนับสายทันที
5. 07:31 คือสายตามกติกา SJ
6. ถ้าลืมแนบรูป ให้ระบบเตือนหรือให้ Admin แก้ย้อนหลังพร้อมเหตุผล

## Daily Work Unit
1. ฝนตกบ่าย 15:00-16:30 และเลิกเพราะไม่คุ้มตั้งของใหม่ หัวหน้าช่างเลือก 1 แรงได้
2. ฝนตกเช้าถึงเที่ยง แล้วเริ่มงานบ่าย หัวหน้าช่างเลือก 1 แรง หรือ 0.5/0.75 ได้ตามความจริง
3. ฝนตกทั้งวัน หัวหน้าช่างเลือก 0.5 หรือ 0 พร้อมเหตุผลได้
4. ปัญหาลูกค้าไม่ให้เข้าพื้นที่ / รอวัสดุ / ลมแรง ต้องใส่เหตุผล

## Advance Payment
1. เบิกได้เฉพาะ 100/200/500/1000
2. ยอดรวมเงินเบิกในรอบ 1-15 หรือ 16-สิ้นเดือน ห้ามเกิน 1,000
3. ถ้าเกิน 1,000 ต้องแจ้งให้ติดต่อเจ้าของ
4. เงินเบิกต้องอนุมัติก่อนบันทึกจ่าย

## Payroll
1. ไม่มี DailyWorkSummary = ไม่คิดแรง
2. สายครบ 3 ครั้ง = หัก 1 แรง
3. หักเงินเบิกเฉพาะรายการ approved/paid
4. Export CSV ต้องอ่านภาษาไทยได้

## Permission
1. Staff เห็นเฉพาะข้อมูลตัวเองในอนาคต
2. Admin เห็นทุกคนและจัดการรายวันได้
3. Super Admin เห็นรายงานทั้งหมดและอนุมัติพิเศษได้
