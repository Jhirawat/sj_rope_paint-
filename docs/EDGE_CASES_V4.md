# EDGE CASES V4

## ลายเซ็นดิจิทัล
- เซ็นว่างแล้วกดบันทึก
- ส่งข้อมูลไม่ใช่ base64 PNG
- ลายเซ็นขนาดเกิน 1MB
- เซ็นซ้ำตำแหน่งเดิม
- storage ไม่มี permission

## QR / Maps
- ไม่มี map_link และไม่มี latitude/longitude
- พิกัดผิดช่วง เช่น latitude > 90
- map_link ไม่ใช่ URL Google Maps
- QR service ภายนอกโหลดไม่ได้

## โทร / LINE
- เบอร์โทรมีขีดหรือเว้นวรรค
- LINE ID มี @ นำหน้า
- LINE ID ว่าง ไม่ควรแสดงปุ่ม LINE

## Check-in
- Browser ไม่อนุญาต Location
- อุปกรณ์ไม่มี GPS
- เช็คอินซ้ำหลายครั้ง
- เช็คอินนอกพื้นที่จริง
- พิกัดว่างแต่กดบันทึก

## แจ้งเตือน
- งานนัดสำรวจพรุ่งนี้หลายรายการ
- งานเริ่มพรุ่งนี้หลายรายการ
- งานเกินกำหนดแต่สถานะเสร็จสิ้น ไม่ควรแสดง
- Timezone ไม่ตรงเครื่อง Server

## พิมพ์ใบงาน A4
- รูปภาพเยอะเกิน 12 รูป
- ไม่มีลายเซ็น
- ไม่มี QR Map
- ข้อความรายละเอียดงานยาวมาก
- Browser ตั้งค่ากระดาษไม่ใช่ A4

## Activity Log
- เปลี่ยนสถานะกลับไปกลับมา
- ผู้ใช้ถูกลบหลังจากเคยเปลี่ยนสถานะ
- สถานะเก่าเป็น null ตอนสร้างใบงาน
