# Edge Cases V5.2

## Request Quote
- ไม่กรอกชื่อ / เบอร์ / บริการ / ประเภทอาคาร / จำนวนชั้น / จังหวัด / อำเภอ / ตำบล / รหัสไปรษณีย์ / รายละเอียดที่อยู่
- เบอร์โทรเป็นตัวอักษร หรือสั้น/ยาวผิดปกติ
- Email กรอกผิดรูปแบบ
- ลิงก์ Google Maps ไม่ใช่ URL
- ใส่พิกัดเกินช่วง lat/lng
- อัปโหลดเกิน 10 รูป
- อัปโหลดไฟล์ไม่ใช่ jpg/jpeg/png/webp
- อัปโหลดไฟล์เกิน 5MB
- อัปโหลดรูปซ้ำในชุดเดียวกัน

## Thai Address
- CDN ใช้ไม่ได้: ต้อง fallback ได้
- เลือกจังหวัดแล้วอำเภอต้องเปลี่ยนตาม
- เลือกอำเภอแล้วตำบลต้องเปลี่ยนตาม
- เลือกตำบลแล้วรหัสไปรษณีย์ต้อง Auto fill
