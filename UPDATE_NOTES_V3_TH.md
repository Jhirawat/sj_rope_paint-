# UPDATE NOTES V3 — SJ ทาสีโรยตัว

V3 รวมสิ่งที่คุยกันล่าสุดทั้งฝั่ง Guest และ Admin โดยเน้นให้เว็บดูน่าเชื่อถือ ใช้งานง่าย และเหมาะกับธุรกิจทาสีโรยตัวจริง

## Guest / หน้าบ้าน
- เพิ่มแนวทาง Hero ใหม่ พร้อมปุ่ม ขอใบเสนอราคา / ดูผลงาน / LINE
- เพิ่ม Floating Contact Bar: โทร / LINE / ขอราคา
- เพิ่มเมนูติดตามสถานะงานด้วยรหัสงาน + เบอร์โทร ไม่ต้อง Login
- เพิ่ม Before / After Showcase ในหน้าแรก
- เพิ่ม Trust Section และ Process Section จาก V2 ต่อเนื่อง
- หน้า Portfolio Detail แสดงข้อมูลโครงการ: ชื่อโครงการ, ประเภทงาน, สถานที่, วันที่ดำเนินงาน
- หน้า Portfolio Detail แยกรูปเป็น ก่อนทำ / ระหว่างทำ / หลังทำ พร้อมสรุปจำนวนรูป
- ฟอร์มขอใบเสนอราคาเน้นข้อมูลจำเป็น: ชื่อ, เบอร์, LINE, ประเภทงาน, พื้นที่, รูปหน้างาน, Maps/พิกัด
- รองรับ TH/EN ต่อเนื่อง และเพิ่มข้อความที่เคยหลุดให้ครบขึ้น

## Admin / หลังบ้าน
- Dashboard V3 เพิ่ม cards: ใบเสนอราคาวันนี้, ยังไม่ติดต่อ, กำลังติดตาม, ใบงานเปิดอยู่, งานกำลังดำเนินการ, งานเสร็จแล้ว, ผลงานที่แสดง, ผู้เข้าชมเว็บ
- เพิ่ม Visitor Logs เพื่อเก็บข้อมูลผู้เข้าชมเบื้องต้น
- Portfolio Management ยกเครื่องใหม่:
  - รูปปกโครงการ 1 รูป
  - รูปก่อนทำ
  - รูประหว่างทำ
  - รูปหลังทำ
  - สรุปจำนวนรูปในหน้ารายการผลงาน
  - ตั้งรูปปกจากรูปในอัลบั้มได้
- ตัดช่องเลขลำดับในฟอร์มผลงานออกจาก UX หลัก ใช้ปุ่มเลื่อนขึ้น/ลงในรายการแทน
- Work Order สามารถสร้าง Project/Portfolio จากใบงานได้
- Backup / Export เพิ่ม CSV สำหรับใบเสนอราคา ลูกค้า ใบงาน
- เพิ่ม service_images, customer_timeline_notes, visitor_logs migration สำหรับต่อยอด Service Gallery, CRM Timeline และ Analytics

## สิ่งที่ยังควรเก็บต่อใน V3.1
- ทำหน้า CRUD Service Gallery เต็มระบบ
- ทำหน้า CRM Timeline สำหรับลูกค้าทุกคน
- ทำ Backup รูปภาพเป็น ZIP จาก storage จริง
- Export Excel/PDF แบบเต็ม ถ้าใช้ package เพิ่ม เช่น PhpSpreadsheet/Dompdf
- ปรับ UI รายละเอียดแต่ละหน้าและทดสอบ Migration บนเครื่องจริง
