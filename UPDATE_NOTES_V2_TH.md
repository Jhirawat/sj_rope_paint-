# Update Notes v2

## Frontend
- ตรวจข้อความหน้าหลักให้รองรับ TH/EN มากขึ้น
- เพิ่มขั้นตอนการทำงานและมาตรฐานความปลอดภัยแบบสองภาษา
- ปรับปุ่มให้ใช้ `display:inline-flex`, `align-items:center`, `line-height:1.25` เพื่อลดปัญหาตัวหนังสือชิดด้านบน

## Admin
- เพิ่ม Work Orders / ใบงาน
- เพิ่มปุ่มสร้างใบงานจากใบเสนอราคา
- ใบงานรองรับพิกัดหน้างานและ Google Maps Link
- เพิ่มรายงานใบงานบน Dashboard
- เพิ่มปุ่มเรียงลำดับแบบ “เลื่อนขึ้น/เลื่อนลง”

## Edge Cases ที่คุมเพิ่ม
- รูปขอใบเสนอราคาเกิน 10 รูป
- พิกัด Latitude/Longitude เกินช่วง
- ใบเสนอราคาเดิมสร้างใบงานซ้ำ ระบบเปิดใบงานเดิมแทน
- ลบบริการที่มีผลงานอ้างอิงไม่ได้
- EN ไม่มีข้อมูล ให้ fallback เป็น TH
