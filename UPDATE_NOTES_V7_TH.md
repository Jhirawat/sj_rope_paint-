# SJ ทาสีโรยตัว V7 - Staff Management Production

## เพิ่มใหม่ใน V7

### Staff Portal สำหรับพนักงาน
- เพิ่ม path `/staff` และ `/staff/dashboard`
- พนักงานเห็นข้อมูลของตัวเองเท่านั้น
- หน้า Mobile First สำหรับอายุ 15-50 ปี
- ปุ่มใหญ่: เข้างาน / ออกงาน / เบิกเงิน / ดูค่าแรง / ดูใบงาน
- Bottom Navigation บนมือถือ

### เข้างาน / ออกงาน
- พนักงานต้องแนบรูปตอนเข้างาน
- บันทึก GPS ได้จากมือถือ
- เวลา 07:31 เป็นต้นไป = สาย
- ออกงานพร้อมแนบรูปหลังเลิกงาน/รูปเก็บของได้

### เงินเบิกฝั่งพนักงาน
- พนักงานขอเบิกผ่านมือถือได้
- เลือก 100 / 200 / 500 / 1000 บาท
- จำกัดไม่เกิน 1,000 บาทต่อรอบจ่าย
- ถ้าเกินให้ติดต่อเจ้าของ
- เมื่อมีการเบิก สร้างแจ้งเตือนให้ Admin ทันที

### Dashboard Admin ใหม่
- เพิ่มกล่องพนักงานทั้งหมด
- เข้างานวันนี้
- สายวันนี้
- คำขอเบิกเงินรออนุมัติ
- กล่องแจ้งเตือนเงินเบิก: ใครเบิก จำนวนเท่าไร และปุ่มจัดการ
- ภาพรวมรอบจ่ายปัจจุบัน

### ปฏิทินใหม่
- ปรับให้เล็กลง ดูทั้งเดือนได้ง่ายขึ้น
- แสดงวันหยุด
- แสดงเงินเบิกแต่ละวัน
- แสดงเริ่มรอบจ่าย / ปิดรอบจ่าย
- มือถือเปลี่ยนเป็น List รายวัน ไม่บีบเป็นตารางเล็กเกินไป

## Route สำคัญ
- `/staff` หน้า Staff Dashboard
- `/staff/advances` เงินเบิกของพนักงาน
- `/staff/payroll` ค่าแรงของพนักงาน
- `/staff/work-orders` ใบงานสำหรับพนักงาน
- `/admin` หลังบ้านเจ้าของ/Admin
- `/admin/staff-calendar` ปฏิทินงาน/วันหยุด/รอบจ่าย

## หมายเหตุ
- Admin route ปรับให้เฉพาะ `admin` และ `super_admin`
- Staff route ใช้ `staff`, `admin`, `super_admin`
- ถ้า Login ด้วย role `staff` จะ redirect ไป `/staff`
