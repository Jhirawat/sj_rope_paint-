# UPDATE NOTES V6 - ระบบพนักงาน / ค่าแรง / เงินเบิก

V6 เพิ่มระบบ Workforce Management สำหรับ SJ ทาสีโรยตัว โดยยึดหลักการทำงานจริงของทีมช่างภาคสนาม:

- เช้าเข้าบริษัท / ประชุมก่อนออกหน้างาน
- 07:31 เป็นต้นไปถือว่าสาย
- ค่าแรงคิดเป็น “แรง” ไม่คิดแบบ OT รายชั่วโมง
- หัวหน้าช่างเป็นผู้สรุปจำนวนแรงตามสภาพหน้างานจริง
- กรณีฝนตก / ปัญหาหน้างาน สามารถสรุปเป็น 1 แรง, 0.75 แรง, 0.5 แรง หรือ 0 แรง พร้อมเหตุผล

## ฟีเจอร์ใหม่

### 1. จัดการพนักงาน
- หัวหน้าช่าง
- ช่างรายวัน
- ผู้ช่วย / กรรมกร
- บัญชี / เลขา / ผู้จัดการ
- เจ้าของกิจการ

### 2. ตั้งค่าค่าแรง
ค่าเริ่มต้น:
- 360
- 400
- 420
- 500
- 550

สามารถเพิ่ม / แก้ไข / ปิดใช้งานค่าแรง และเลือกให้พนักงานเป็นรายคนได้

### 3. เข้างาน / ออกงาน
- บันทึกเข้างาน
- บันทึกออกงาน
- แนบรูปเข้างาน / ออกงาน
- จุดเข้างาน: บริษัท / หน้างาน
- บันทึก GPS
- เข้าหลัง 07:30 ถือว่าสาย
- สายครบ 3 ครั้ง สร้างใบเตือนและหัก 1 แรงอัตโนมัติ

### 4. สรุปแรงประจำวัน
หัวหน้าช่าง / Admin สรุปได้:
- 1 แรง
- 0.75 แรง
- 0.5 แรง
- 0 แรง

สถานะ:
- เต็มวัน
- ครึ่งวัน
- หยุดเพราะฝน
- ปัญหาหน้างาน
- ลา
- ขาด

### 5. เงินเบิก
- ตัวเลือก 100 / 200 / 500 / 1000
- กันเบิกเกิน 1,000 บาทต่อรอบจ่าย
- ถ้าต้องการมากกว่า 1,000 ต้องติดต่อเจ้าของ
- สถานะ: pending / approved / rejected / paid

### 6. รอบจ่ายค่าแรง
- รอบ 1: วันที่ 1-15
- รอบ 2: วันที่ 16-สิ้นเดือน

สูตร:
จำนวนแรง × ค่าแรงรายวัน = ค่าแรงรวม
ค่าแรงรวม - หักสาย/ใบเตือน - เงินเบิก = ยอดสุทธิ

### 7. ปฏิทินงาน / วันหยุด / รอบจ่าย
- วันหยุดนักขัตฤกษ์
- วันหยุดบริษัท
- กิจกรรมงาน
- วันเบิกเงิน
- รอบจ่ายค่าแรง

- เข้างาน/แรง

## Edge Cases V6
- เข้างานซ้ำวันเดียวกันไม่ได้
- ออกงานไม่ได้ถ้ายังไม่ได้เข้างาน
- สายตั้งแต่ 07:31
- สายครบ 3 ครั้งหัก 1 แรง + ใบเตือน
- เบิกเกิน 1,000 ต่อรอบไม่ได้
- หัวหน้าช่างต้องใส่เหตุผลเมื่อเลือกหยุดเพราะฝน/ปัญหาหน้างาน
- ลบพนักงานที่มีประวัติไม่ได้ ให้ปิดใช้งานแทน
